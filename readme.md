# The Future We Were Promised

A pure front-end web prototype dedicated to exploring and expressing the nostalgic beauty of early 2000s internet aesthetics. This project is a visual love letter to the optimistic digital landscapes that shaped a generation.

## 🌌 About the Project
"The Future We Were Promised" is an interactive playground built to capture the distinct textures, colors, and comforting visual languages of the late 90s and 2000s. It is designed to be a highly tactile, satisfying experience where each aesthetic shifts the entire atmosphere of the page—complete with responsive canvas elements, custom cursors, and thematic synthesizer soundscapes.

## ✨ Featured Genres
Explore the curated collection of eras that defined an optimistic future:

*   **Frutiger Aero** – The iconic utopia of eco-optimism, glossy glass textures, vibrant blue skies, and clear water.
*   **Dorfic** – A cozy, tactile sanctuary inspired by translucent orange plastics, amber glows, and friendly product curves.
*   **Dark Aero** – The edgy, cyber-futuristic abyss of late 2000s gaming, deep metallic blacks, and neon green radar grids.
*   **Clean Core** – A gentle, sterile dreamscape filled with soft pastel gradients, soap foam, and pure domestic peace.
*   **Frutiger Metro** – An energetic burst of vector art, flat color blocks, and indie-pop urban music aesthetics.

*🔮 More nostalgic genres coming soon...*

## 🛠️ Built With
*   Pure HTML5 & Vanilla JavaScript
*   Tailwind CSS (Styling)
*   Canvas API (Interactive Backgrounds)
*   Web Audio API (Thematic Sound Synth)
